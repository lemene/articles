# 安装

## 概述
本文收集如何在单机安装多个Python环境的方法。
+ Python2和Python3并存，
+ 没有管理员权限，第三方库只能安装在本地。
+ 分类第三方库，不同的库安装在不同位置。


## 安装位置
pip命令：使用`--user`和`PYTHONUSERBASE`配合使用。

    pip install *package* --user
